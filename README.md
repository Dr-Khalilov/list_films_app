## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Running the app

```bash
$ ./start-dev.sh
```

## Stopping the app

```bash
$ CTRL + C or ./stop.sh
```

## Clear all docker containers, images and volumes

```bash
$ ./prune.sh
```

http://localhost:8081
